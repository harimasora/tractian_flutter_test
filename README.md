## Instalation

To install this project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/harimasora/tractian_flutter_test.git
    ```
2. Install the dependencies:
    ```sh
    flutter pub get
    ```
3. Run:
    ```sh
    flutter run
    ```
